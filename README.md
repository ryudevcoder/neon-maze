# Neon-Maze

Um jogo de labirinto em 2D de alta performance com estética neon, inspirado no clássico Pac-Man.

## 🚀 Funcionalidades
- **Movimentação em Grade:** Precisão clássica de arcade.
- **IA Diferenciada:** Fantasmas com comportamentos de Perseguição, Emboscada e Aleatório.
- **Sistema de Wraparound:** Túneis laterais funcionais.
- **Power-Ups:** Reversão de vulnerabilidade dos inimigos.
- **Combo Multiplier:** Bônus para coleta rápida de pontos.
- **Progressão:** Dificuldade crescente a cada nível.

## 🛠️ Tecnologias
- HTML5 Canvas
- JavaScript Moderno (ES6+)
- CSS3 Custom Properties & Animations

## 📖 Como Rodar
Como o projeto utiliza Módulos ES6, é necessário rodar através de um servidor local:
1. Use a extensão **Live Server** no VS Code.
2. Ou rode `npx serve` na pasta raiz.
